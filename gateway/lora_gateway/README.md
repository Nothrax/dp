# LoRa Gateway
Gateway for LoRaWAN.

# Dependencies
[spdlog](https://github.com/gabime/spdlog) >= 1.12.0
[boost](https://www.boost.org/) >= 1.79.0
[wiringPi](http://wiringpi.com/) >= 2.46.0

# Build
```
mkdir _build
cd _build
cmake ..
make
```

# Arguments
* `--config|-c <string>` path to json configuration file
